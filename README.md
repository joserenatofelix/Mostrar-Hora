
# <h1 align="center"> MOSTRA A HORA E CALCULA SUA IDADE </h1>

## <b>O projeto</b> </br>
Projeto, bem simples mas muito intuitivo que mostra a hora online e na parte de baixo calcula a idade através do ano de nascimento.

## Tecnologias

- HTML5</br>
- CSS3</br>
